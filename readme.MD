# README


## Configure varibles in the template
```
{{ SENDERNAME }} = Senders first and last name
{{ SENDER }} = Senders emails address that auths with mail server.
{{ SPOOFED }} = The email that is shown as sent from.
{{ SENDERPOSITION }} = The senders position.
{{ SENDERDEPARTMENT }} = The senders department.
{{ RECIPIENT }} = The recipients email address
{{ RECIPIENTNAME}} = The recipients first and last name.
{{ RECIPIENTPOSITION }} = The recipients position we don't have these
{{ uuid }} = The unique id for the email whihc must be passed in a link like https://limerock.xyz?=email_id={{ uuid }}
```
